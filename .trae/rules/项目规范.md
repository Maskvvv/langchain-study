# 项目规范

## 包管理

- 本项目使用 [uv](https://docs.astral.sh/uv/) 作为 Python 包管理工具
- 添加依赖：`uv add <package>`
- 移除依赖：`uv remove <package>`
- 安装项目所有依赖：`uv sync`
- 运行脚本：`uv run python <script.py>`
- **禁止**使用 `pip install` 直接安装依赖，所有依赖必须通过 `uv add` 添加到 `pyproject.toml`

## Python 版本

- 项目要求 Python >= 3.13
- 使用 `uv python install` 安装 Python 版本
- 使用 `uv python pin <version>` 固定项目 Python 版本

## 代码风格

- 遵循 PEP 8 规范
- 文件编码统一使用 UTF-8
