# 第一章：LangChain 概览与环境搭建

## 1.1 LangChain 是什么？

LangChain 是一个开源框架，用于构建基于大型语言模型（LLM）的应用程序。它提供了一套标准化的接口和组件，让开发者能够轻松地将 LLM 与外部数据源、工具和计算能力连接起来。

### 核心设计哲学

LangChain 的设计遵循以下原则：

1. **组件化**：每个功能都是独立的组件，可以自由组合
2. **抽象化**：提供统一的接口，屏蔽底层实现差异
3. **可扩展**：支持自定义组件，满足特定需求

## 1.2 LangChain 核心架构

```
┌─────────────────────────────────────────────┐
│                  Application                │
├─────────────────────────────────────────────┤
│              Agent（智能体）                  │
│         ┌──────────┬──────────┐             │
│         │   Tool   │  Memory  │             │
│         └──────────┴──────────┘             │
├─────────────────────────────────────────────┤
│              Chain（链式调用）                │
│    ┌─────────┬──────────┬──────────┐        │
│    │ Prompt  │   LLM    │  Output  │        │
│    │Template │  Model   │  Parser  │        │
│    └─────────┴──────────┴──────────┘        │
├─────────────────────────────────────────────┤
│           Retrieval（检索增强）               │
│    ┌─────────┬──────────┬──────────┐        │
│    │Document │   Text   │  Vector  │        │
│    │ Loader  │ Splitter │  Store   │        │
│    └─────────┴──────────┴──────────┘        │
├─────────────────────────────────────────────┤
│            Model I/O（模型交互）              │
│    ┌─────────┬──────────┬──────────┐        │
│    │ Prompt  │ Language │  Output  │        │
│    │   I/O   │  Model   │  Parser  │        │
│    └─────────┴──────────┴──────────┘        │
└─────────────────────────────────────────────┘
```

### 六大核心模块

| 模块 | 说明 |
|------|------|
| **Model I/O** | 与大模型交互的接口，包括 Prompt 管理、模型调用、输出解析 |
| **Retrieval** | 与外部数据交互，包括文档加载、切分、嵌入、向量存储与检索 |
| **Chains** | 将多个组件串联成链式调用，实现复杂工作流 |
| **Memory** | 在链/Agent 调用之间持久化应用状态 |
| **Agents** | 根据用户输入动态决定调用哪些工具和操作 |
| **Callbacks** | 日志记录、流式输出等横切关注点 |

## 1.3 LangChain 生态

- **LangChain**：核心框架，提供基础抽象和组件
- **LangGraph**：基于状态图的 Agent 构建框架（当前主推的 Agent 方案）
- **LangSmith**：调试、测试和监控平台
- **LangServe**：将 Chain 部署为 REST API

## 1.4 环境搭建

### 安装依赖

```bash
# 使用 uv 安装（推荐）
uv add langchain langchain-openai langchain-community langgraph

# 需要向量数据库时
uv add chromadb faiss-cpu
```

### 配置 API Key

复制 `.env.example` 为 `.env`，填入你的 API Key：

```bash
cp .env.example .env
```

### 验证安装

运行 `hello_langchain.py` 验证环境是否正常。
